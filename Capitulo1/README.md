# Capítulo 1 — Abriendo el apetito

## ¿Por qué usar Python?
Cuando se trabaja mucho con computadoras, es común querer automatizar tareas como buscar y reemplazar en muchos archivos, renombrar fotos, crear una pequeña base de datos, una aplicación sencilla o incluso un juego.  
Python facilita todas estas tareas.

## Ventajas frente a otros lenguajes
- Es más rápido de desarrollar que C, C++ o Java porque no requiere compilación.
- Más potente que un script de shell o archivos batch.
- Compatible con Windows, macOS y Linux.
- Ofrece más comprobación de errores y tipos de datos de alto nivel.
- Permite resolver una gran variedad de problemas de forma sencilla.

## Módulos y reutilización
Python permite dividir un programa en módulos reutilizables.  
Incluye una amplia biblioteca estándar con:
- Manejo de archivos  
- Sockets  
- Interfaces gráficas (como Tk)  
- Diversas utilidades listas para usar

## Un lenguaje interpretado
- No requiere compilar ni enlazar.  
- Puede usarse de forma interactiva para hacer pruebas o experimentar con el lenguaje.  
- También funciona como una calculadora útil para tareas rápidas.

## Código más corto y legible
Los programas en Python suelen ser más cortos porque:
- Sus tipos de alto nivel permiten expresar operaciones complejas en una sola línea.  
- La estructura del código se basa en sangría en lugar de llaves.  
- No es necesario declarar variables.

## Extensible
Si conoces C, puedes añadir funciones o módulos al intérprete para:
- Ejecutar operaciones críticas más rápido  
- Conectar Python con bibliotecas externas  

También es posible integrar el intérprete dentro de aplicaciones escritas en C.

## Origen del nombre
Python toma su nombre del programa de la BBC “Monty Python’s Flying Circus”.
